Pedigree Analysis Pipeline

This vignette demonstrates a complete workflow for pedigree analysis using the visPedigree package. We will cover basic statistics, inbreeding monitoring, population structure analysis, and advanced ancestral tracking using the implemented gene flow and partial inbreeding algorithms.

1. Setup and Data Preparation

First, we load the package and prepare a sample pedigree. We will use a simulated pedigree simple_ped included in the package, but for this demonstration, we’ll simulate a slightly more complex structure with inbreeding events to showcase the features.

library(visPedigree)
library(data.table)

# Simulate a pedigree with 5 generations
# Foundation: 2 males (S1, S2), 2 females (D1, D2)
ped_data <- data.table(
  Ind = c("S1", "S2", "D1", "D2", 
          "G1_1", "G1_2", "G1_3", "G1_4",
          "G2_1", "G_Inbred", 
          "G3_1"),
  Sire = c(NA, NA, NA, NA, 
           "S1", "S1", "S2", "S2", 
           "G1_1", "G1_1", 
           "G2_1"),
  Dam = c(NA, NA, NA, NA, 
          "D1", "D2", "D1", "D2", 
          "G1_3", "G1_3", # G_Inbred is G1_1 x G1_3 (Full Sibs if S1,D1? No. G1_1=S1xD1, G1_3=S2xD1. Half-sibs)
          "G_Inbred"),
  Sex = c("male", "male", "female", "female",
          "male", "male", "female", "female",
          "male", "female",
          "male"),
  BirthYear = c(2010, 2010, 2010, 2010,
                2012, 2012, 2013, 2013,
                2015, 2016,
                2019)
)

# Tidy the pedigree (sorts, adds generations, checks integrity)
tped <- tidyped(ped_data)

Effective Population Size (Ne)

Estimate the effective population size (\(N_e\)), which is a key metric for monitoring genetic diversity. visPedigree provides three methods: based on coancestry ("coancestry", the default), rate of inbreeding ("inbreeding"), or demographic counts ("demographic").

You can calculate \(N_e\) grouped by year or generation:

# Estimate Ne per cohort (grouped by BirthYear)
ne_est <- pedne(tped, by = "BirthYear", method = "inbreeding")
print(ne_est)
#> Tidy Pedigree Object
#>    Cohort     N  MeanF    DeltaF MeanECG      Ne
#>     <num> <int>  <num>     <num>   <num>   <num>
#> 1:   2015     1 0.1250 0.1250000       2 4.00000
#> 2:   2016     1 0.1250 0.1250000       2 4.00000
#> 3:   2019     1 0.3125 0.1708438       3 2.92665

Alternatively, you can evaluate the overall \(N_e\) for a specific subset (e.g., the last generation) by omitting the by parameter:

# Get the IDs of the latest generation
latest_gen_ids <- tped[Gen == max(Gen), Ind]

# Calculate overall Ne for this specific candidate group using the default coancestry method
overall_ne <- pedne(tped, cand = latest_gen_ids)
#> No 'by' parameter specified. Calculating Ne for the entire dataset. Grouping by a cohort or year variable is often more biologically meaningful.
#> Warning in validate_and_prepare_ped(ped): Removed 10 records with duplicate
#> Ind/Sire/Dam IDs.
print(overall_ne)
#>    Cohort     N NSampled DeltaC    Ne
#>    <char> <int>    <int>  <num> <num>
#> 1:    All    11       11     NA    NA

Gene Flow / Ancestry Proportions

If our founders came from different origins (e.g., Line A vs Line B), we can trace their proportions.

# Assign arbitrary labels to founders for demonstration
tped_dated <- copy(tped)
tped_dated[Ind %in% c("S1", "D1"), Origin := "LineA"]
tped_dated[Ind %in% c("S2", "D2"), Origin := "LineB"]

# Calculate proportion of LineA vs LineB in all individuals
ancestry <- pedancestry(tped_dated, labelvar = "Origin")
print(tail(ancestry))
#>         Ind LineA LineB

6. Advanced: Partial Inbreeding

Partial inbreeding (\(F_{ij}\)) allows us to decompose an individual’s total inbreeding coefficient into the specific contributions from each ancestor. This answers: “Who is responsible for the inbreeding of this animal?”

# Let's analyze the inbreeding of the last individual (G3_1)
target_ind <- "G3_1"
total_f <- tped[Ind == target_ind, f]
cat("Total Inbreeding for", target_ind, ":", total_f, "\n")
#> Total Inbreeding for G3_1 : 0.3125

# Calculate partial inbreeding from top ancestors
partial_f <- pedpartial(tped, ancestors = c("S1", "D1", "S2", "D2"), top = 5)
#> Calculating partial inbreeding for 4 ancestors...

# Extract result for the target individual
res <- partial_f[Ind == target_ind]
print(res)
#>       Ind    D1    D2      S1      S2
#>    <char> <num> <num>   <num>   <num>
#> 1:   G3_1 0.125     0 0.03125 0.03125

Summary

visPedigree provides a full suite of tools for: 1. Demography: Stats and intervals. 2. Health: Inbreeding and diversity monitoring. 3. Structure: Ancestral contributions and gene flow. 4. Visualization: (See vignette("draw-pedigree") for plotting details).

These tools enable efficient management of breeding programs and conservation genetics.
